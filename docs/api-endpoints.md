# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users/id`

### Bnb

- `GET /api/bnbs`
- `GET /api/bnbs/:id`


### Reviews

- `POST /api/reviews`
- `PATCH /api/reviews/:id`
- `DELETE /api/reviews/:id`

### Bookings

- `POST /api/bookings`
- `DEL /api/bookings/:id`
