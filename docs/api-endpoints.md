# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Bnb

- `POST /api/bnbs/:id`
- `POST /api/bnbs`
- `PATCH /api/bnbs/:id`
- `DELETE /api/bnbs/:id`

### Reviews

- `GET /api/reviews/:id`
- `POST /api/reviews`
- `PATCH /api/reviews/:id`
- `DELETE /api/reviews/:id`
